Real-time AI Driver Drowsiness Detection A web-based computer vision application designed to prevent road accidents by detecting driver fatigue through facial landmark analysis.

-> Key Features
Eye Aspect Ratio (EAR) Monitoring: Detects prolonged eye closure.

Yawn Detection: Tracks mouth movements for signs of exhaustion.

Instant Audio Alerts: Real-time warnings when drowsiness is detected.

Web-First: No installation required; runs directly in the browser via webcam.
